# 🔐 AI Cyber Threat Detector

An AI-powered web application that analyzes messages and user profile data to detect potential cyber threats such as harassment, scams, and suspicious behavior.

---

## 🚀 Features

- 🤖 AI-based message analysis using Google Gemini API  
- 👤 Profile risk evaluation (followers, following ratio, activity)  
- ⚠️ Threat classification: Safe / Suspicious / Dangerous  
- 📊 Risk scoring system  
- 💡 Suggested user actions (Block, Report, Stay cautious)  
- 🌐 Fully deployed (Frontend + Backend)

---

## 🧠 How It Works

1. User enters a message and profile details  
2. Frontend sends data to backend API  
3. Backend:
   - Analyzes profile risk  
   - Sends message to Gemini API  
   - Combines AI + behavioral signals  
4. Returns:
   - Risk score  
   - Threat level  
   - Explanation  
5. Frontend displays result

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express  
- **AI Model:** Google Gemini API  
- **Deployment:** Render (Backend), Vercel (Frontend)
